# Pryv account backup — sample webapp (demo)

This is a static deploy of [`pryv/pryv-account-backup-webapp`](https://github.com/pryv/pryv-account-backup-webapp) on GitHub Pages, served from [https://pryv.github.io/pryv-account-backup-webapp/](https://pryv.github.io/pryv-account-backup-webapp/). It is a **reference demo only** — operators are expected to fork the repo and host their own instance under a domain the subject already trusts (same-origin as the Pryv API).
